# Penguin Fashion
This is a fashion Website
🚀 Live Demo

🔗 https://penguin-fashion-tawny.vercel.app/

📌 Features
Fully responsive design (Mobile, Tablet, Desktop)
Modern and clean fashion UI
Responsive navigation bar
Hero section with strong visual hierarchy
Product showcase sections
Reusable Tailwind utility classes
Optimized layout without external CSS frameworks
🛠️ Technologies Used
HTML5
Tailwind CSS
Vercel (for deployment)
📁 Project Structure
penguin-fashion/
│── index.html
│── images/
│── icons/
│── tailwind.config.js
│── README.md
└── .gitattributes
⚙️ Setup & Run Locally
Clone the repository:
git clone https://github.com/istewakhassantewak/penguin-fashion.git
Navigate into the project folder:
cd penguin-fashion
Open index.html in your browser.

No build tools required — pure HTML + Tailwind setup.
